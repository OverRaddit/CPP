# 클래스 내 생성자

- 클래스가 처음 생성될 때, 멤버변수들은 쓰레기값이 들어있다.
	보통 변수를 선언하고 적절히 초기화해야 하는데 이를 하나의 작업으로 처리해주는 것이 생성자이다.
	객체를 생성할때 자동으로 호출되는 함수이며, 값을 초기화해주는 작업만을 하기때문에
	리턴값이 없다.

```C++
Contact contact = Contact();		// 지역변수 생성
Contact *p_contact = new Contact();	// new는 말록처럼 동적할당 후 주소를 반환한다.
```

- 생성자로 초기화된 변수는 지역변수인가, 동적으로 할당받은 변수인가?
	=> 생성자와 동적할당은 관련이 없다. 잘못된 질문임.
	=> 생성자는 클래스를 인스턴스로 만들어준다. 동적으로 할당받고싶으면 new를 앞에 붙이면 된다.
